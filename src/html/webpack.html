This is a basic example of a critical CSS approach

This is a simple example of my personal approach to critical CSS.

Basically I write my SASS/SCSS like I always do but I add simple annotations to the rules that should be included in the critical CSS.

The Gulp task then first processes the SASS file and then splits up the CSS files into 2 CSS files: some.css and some-critical.css. Then the URL's in the some-critical.css file are updated to their new relative paths before being inlined in the head of the site. The rest of the CSS is loaded with some inline JS.

This removes any blocking action of CSS from the site.

For the sake of no-Javascript support, we add the some.css styleshheet the old fashioned way in noscript tags right before the </body> tag.
